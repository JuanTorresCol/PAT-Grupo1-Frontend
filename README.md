AtilanoPadel

Aplicación web estática para la gestión de reservas de pistas de pádel.
Incluye funcionalidades diferenciadas para usuarios y administradores.

Descripción

AtilanoPadel es una web orientada a la reserva de pistas de pádel.
Permite a los usuarios consultar disponibilidad, reservar horarios y gestionar sus sesiones.

El sistema distingue entre dos tipos de perfil:

Usuario (USER) → puede ver pistas y reservar
Administrador (ADMIN) → gestiona pistas, reservas y usuarios
Acceso

La aplicación está desplegada en GitHub Pages y se puede acceder desde:

https://juantorrescol.github.io/PAT-Grupo1-Frontend/home.html

Estructura de la web

Las principales secciones son:

Inicio → página principal
Pistas → listado de pistas disponibles
Login → acceso de usuarios
Reservas → gestión de reservas
Panel de usuario/admin → opciones según rol
